# ⚖️ AI Legal Aid — RTI & FIR Generator

AI Legal Aid is a full-stack web application that helps users generate legal documents like RTI applications, FIR reports, and Consumer Complaint letters in both English and Hindi using AI-powered document generation.

This project is built with a modern tech stack using React, Vite, Node.js, Express, and MongoDB.

---

# 🚀 Features

* 📝 Generate RTI Applications
* 🚔 Create FIR Complaint Drafts
* ⚖️ Consumer Complaint Letter Generator
* 🌐 Multi-language Support (English & Hindi)
* 🤖 AI-based Legal Draft Assistance
* 📂 Clean and Responsive UI
* 🔒 Backend API Integration
* 💾 MongoDB Database Support

---

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone <your-repo-link>
cd AI-Legal
```

---

## 2️⃣ Install Dependencies

### Install Root Dependencies

```bash
npm install
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the `server` folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection
OPENAI_API_KEY=your_api_key
```

---

# ▶️ Run the Project

## Start Backend

```bash
cd server
npm run dev
```

## Start Frontend

```bash
cd client
npm run dev
```
